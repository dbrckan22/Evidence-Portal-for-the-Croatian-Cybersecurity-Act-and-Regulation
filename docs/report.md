# Završno izvješće projekta  
## Portal za dokaze prema Zakonu o kibernetičkoj sigurnosti (NIS2)

---

## Svrha projekta

Cilj projekta bio je razviti web aplikaciju za centralizirano praćenje zakonskih obveza i prikupljanje dokaza o usklađenosti s hrvatskim Zakonom o kibernetičkoj sigurnosti i NIS2 direktivom.

---

## Opis rješenja

Sustav je implementiran kao web aplikacija s frontend i backend dijelom te SQL Server bazom podataka.  
Backend osigurava API i obradu podataka, dok frontend omogućuje pregled obveza, unos dokaza i prikaz statusa usklađenosti.

---

## Model i funkcionalnosti

Model podataka obuhvaća organizacije, korisnike, obveze i dokaze, uz mogućnost formiranja audit paketa za izvoz.  
Implementirane su funkcionalnosti pregleda obveza, dodavanja dokaza i generiranja ZIP paketa za reviziju.

---

## Zaključak

Razvijena aplikacija omogućuje jednostavnije upravljanje dokazima i bolju pripremu za inspekcije, te predstavlja dobru osnovu za daljnji razvoj sustava za upravljanje kibernetičkom sigurnošću.
